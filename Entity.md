# 의미
- 실제 세계에서 존재하거나 개념적으로 구분할 수 있는 Object 또는 사물

# 특징
- Attribute의 Set으로 표현됨

# Entity(Database)
- Database에서 저장하고 관리해야 할 Data를 나타냄

# 예시
1. 학생(Entity) : 학생에 대한 정보를 저장하기 위한 Entity
    - Attribute들
        - 학번
        - 이름
        - 전공
        - 나이
    - Primary Key
        - 학번
2. 제품(Entity) : 제품에 대한 정보를 저장하기 위한 Entity
    - Attribute들
        - 제품 번호
        - 제품 이름
        - 제품 가격
    - Primary Key
        - 제품 번호
